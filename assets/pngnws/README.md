# PNGNWS preview assets

This folder stores mirrored preview images used by the **PNGNWS Outlook** tab in the dashboard.

These files are refreshed automatically by the GitHub Actions workflow:
- `.github/workflows/update-pngnws-preview-assets.yml`

The current mirrored preview set includes:
- 3-month rainfall outlook (PNG domain)
- 3-month rainfall outlook (global)
- 31-day rainfall preview (PNG domain)
- 31-day temperature preview (PNG domain)
- Himawari IR1S satellite preview
- GSMaP daily rainfall preview

These files are intended only as quick-look public-facing previews for the dashboard.
